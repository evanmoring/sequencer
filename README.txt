A small library for audio signal processing and generation.

Features:
* sine and square wave generation
* noise generation
* pitch bending
* scale support
* time and beat domain sequencing
* tools for saving and loading sequences
* demonstration of a novel compression method (sigmoid compressor)
* envelopes
* FFT-based frequency filtering
* frequency and time domain visualization
* formant support for voice synthesis
* analysis tools for frequency sweeps

see example/ for usage
pip3 install -r requirements.txt
python3 -m example.example

drum_machine_example is a sketch of an example future project. It isn't supported by the requirements.txt so libraries will have to be imported manually.
